# Portfolio - Personal website

## Developing

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npx svelte-kit dev 

# add `--host` to expose the app on your LAN
npx svelte-kit dev --host
```

## Building

```bash
npx svelte-kit build
```

## Deploying

In the build folder:
```bash
node index.js
```

The app runs on port 3000.
